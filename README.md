# Mobile Directory Management System

## Project Description

This is a Python-based Mobile Directory Management System that helps users manage contact details efficiently. It allows users to store, search, update, and delete contact information easily.

---

## Features

* Add new contacts
* Search contacts
* Update existing contacts
* Delete contacts
* Store data using CSV file
* Export contacts to PDF

---

## Technologies Used

* Python
* CSV (for data storage & file handling)
* IDLE (Python IDE for development)

---

## Project Structure

```
Mobile-Directory-System/
│── main.py
│── contacts.csv
│── README.md
```

## How to Run

1. Install Python on your system
2. Open the project folder
3. Run the file using:
